This folder contains 4 python notebook files.

FLBERT contains code for federated learning with BERT model using depression dataset.
FLDistillBert.ipynb contains code for federated learning with Distill BERT model using depression dataset.
FLRoberta.ipynb contains code for federated learning with Roberta BERT model using depression dataset.
BERT_Non_Federated.ipynb contains code for non - federated learning with BERT model using depression dataset.

These notebooks can be run without any prequisites using google colab.
